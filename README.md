# 💰 Personal Finance Dashboard

A responsive and user-friendly **Personal Finance Dashboard** developed using **HTML, CSS, and JavaScript**. This project helps users manage their personal finances by tracking income, expenses, balance, and transaction history through a clean and interactive interface.

---

## 📌 Project Overview

Managing personal finances is an important part of everyday life. This dashboard provides a simple way to record financial transactions, calculate available balance, and maintain a history of income and expenses.

The application is designed with a modern and responsive interface, making it easy to use on desktops, tablets, and mobile devices.

---

## 🚀 Features

- Add new income transactions
- Add new expense transactions
- Automatic balance calculation
- View transaction history
- Delete transactions
- Responsive dashboard design
- Clean and intuitive user interface
- Real-time financial summary

## 📖 How It Works

1. Enter a transaction description.
2. Enter the transaction amount.
3. Select Income or Expense.
4. Click **Add Transaction**.
5. The dashboard automatically updates:
   - Total Income
   - Total Expenses
   - Available Balance
6. View all transactions in the history section.
7. Delete any transaction whenever required.
